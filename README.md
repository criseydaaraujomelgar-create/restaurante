# 🍽️ API del Restaurante

API REST para gestionar un restaurante con autenticación JWT, control de roles y documentación interactiva.

## 📋 Descripción

Este proyecto es una API RESTful desarrollada con Node.js, Express y PostgreSQL que permite gestionar:
- **Usuarios**: Registro, login y perfiles con autenticación JWT
- **Mesas**: CRUD completo con control de roles (admin/cliente)
- **Reservaciones**: Sistema completo de reservaciones con validaciones

## 🛠️ Tecnologías Utilizadas

- **Node.js** - Entorno de ejecución
- **Express** - Framework web
- **PostgreSQL** - Base de datos
- **JWT** - Autenticación con tokens
- **bcrypt** - Encriptación de contraseñas
- **Swagger** - Documentación interactiva
- **nodemon** - Desarrollo con hot-reload
